# TypeScript 安装与配置

2. TypeScript 入门教程 [https://ts.xcatliu.com/introduction/what-is-typescript.html]

3. 安装及配置：

```
# 全局安装
  npm install -g typescript
# 查看版本
  tsc --version
# .ts文件编译成.js文件
  tsc xxx.ts
```
